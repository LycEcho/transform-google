## 参数

| 名称	| 类型	| 必填	| 说明 |
| :------------: | :------------: | :------------: | :------------: |
| content	| string	| 是	| 文章内容
| from	| string	| 是	| 翻译前语言      【zh-CN=中文（简体）】【zh-TW=中文（繁体）】【ru=俄语】【en=英语】【de=德语】【ko=韩语】【fr=法语】【ja=日语】
| to	| string	| 是	| 要翻译成什么语言
| token	| string	| 是	| 接口密钥可在购买接口后获取
